Rules

구조 규칙

  • 엔티티명은 단수형 기준
  • DTO는 *DTO
  • 서비스 구현체는 *ServiceImpl
  • 컨트롤러는 도메인 기준 하나씩 분리
  • Repository는 aggregate root 또는 주요 테이블 기준으로 구성
