기능 범위

회원, 판매자, 예약, 문의, 혜택 흐름을 역할별로 읽기 쉽게 나눈 문서다.

회원 판매자 숙소/객실 예약/결제 리뷰/찜 문의 이벤트/쿠폰/마일리지 관리자

Member

회원 기능

01
회원가입

이메일, 이름, 전화번호 기반 회원 등록과 기본 권한 ROLE_USER 부여.

02
로그인

LOCAL 로그인 지원, 소셜 로그인 확장 가능, 리프레시 토큰 관리.

03
회원 상태 관리

활성, 휴면, 차단, 탈퇴 상태 관리.

Seller

판매자 기능

  • 판매자 신청 시 사업자 정보를 제출하고 승인 대기 상태를 만든다.
  • 관리자는 승인, 반려, 운영 중지 처리를 할 수 있다.

Lodging

숙소 / 객실 기능

숙소

  • 숙소명, 유형, 지역, 주소, 설명 등록
  • 체크인/체크아웃 시간 입력
  • 수정, 비노출, 운영 중지
  • 이미지 등록 및 정렬 순서 관리

객실

  • 객실명, 유형, 설명, 최대인원, 가격 등록
  • 예약 가능/불가 상태 관리
  • 운영 중지 처리

Booking

예약 / 결제 기능

예약

  • 객실, 체크인, 체크아웃, 인원, 요청사항 입력
  • 가격, 할인금액, 총액 계산 후 저장
  • 상태: PENDING, CONFIRMED, CANCELED, COMPLETED, NO_SHOW
  • 취소 시 취소 주체와 취소 시각 기록

결제

  • 예약 기준 결제 생성
  • 결제수단, PG 정보 저장
  • 상태: READY, PAID, FAILED, CANCELED, PARTIAL_CANCELED, REFUNDED

Review

리뷰 / 찜 기능

  • 리뷰는 예약당 1회 작성
  • 평점, 내용, 이미지 등록
  • 리뷰 운영 상태: 노출, 숨김, 신고
  • 찜 등록, 찜 해제, 찜 목록 조회

Inquiry

문의 기능

문의는 채팅형 구조를 기준으로 설명한다.
  • 문의방 생성: 제목, 문의 유형, 관련 숙소/예약 연결
  • 문의 메시지 송수신: 회원, 판매자, 관리자 간 메시지, 읽음 여부 관리
  • 문의 상태: 접수, 답변 완료, 종료, 차단

Benefit

이벤트 / 쿠폰 / 마일리지 기능

  • 이벤트 등록, 수정, 숨김
  • 쿠폰 생성, 발급, 비활성화
  • 회원 쿠폰 발급, 사용, 만료, 취소
  • 마일리지 적립, 사용, 사용취소, 만료, 조정

Admin

관리자 기능

  • 판매자 승인 관리
  • 회원 상태 관리
  • 이벤트/쿠폰 관리
  • 문의 모니터링
  • 감사 로그 조회
