# TripZone 기능 명세서

## 1. 회원 기능

### 1.1 회원가입

- 이메일, 이름, 전화번호 기반 회원 등록
- 기본 권한 `ROLE_USER` 부여

### 1.2 로그인

- LOCAL 로그인 지원
- 소셜 로그인 확장 가능
- 리프레시 토큰 관리

### 1.3 회원 상태 관리

- 활성, 휴면, 차단, 탈퇴 상태 관리

## 2. 판매자 기능

### 2.1 판매자 신청

- 사업자 정보 제출
- 관리자 승인 대기 상태 생성

### 2.2 판매자 승인 관리

- 승인
- 반려
- 운영 중지

## 3. 숙소 기능

### 3.1 숙소 등록

- 숙소명, 유형, 지역, 주소, 설명 등록
- 체크인/체크아웃 시간 입력

### 3.2 숙소 관리

- 수정
- 비노출
- 운영 중지

### 3.3 숙소 이미지 관리

- 대표/일반 이미지 등록
- 정렬 순서 관리

## 4. 객실 기능

### 4.1 객실 등록

- 객실명, 유형, 설명, 최대인원, 가격 등록

### 4.2 객실 관리

- 예약 가능/불가
- 운영 중지

## 5. 예약 기능

### 5.1 예약 생성

- 객실, 체크인, 체크아웃, 인원, 요청사항 입력
- 가격, 할인금액, 총액 계산 후 저장

### 5.2 예약 상태 관리

- 대기
- 확정
- 취소
- 숙박완료
- 노쇼

### 5.3 예약 취소

- 취소 시 취소 주체, 취소 시각 기록

## 6. 결제 기능

### 6.1 결제 생성

- 예약 기준 결제 생성
- 결제수단 및 PG 정보 저장

### 6.2 결제 상태 처리

- 결제완료
- 실패
- 취소
- 부분취소
- 환불

## 7. 리뷰 기능

### 7.1 리뷰 작성

- 예약당 1회 작성
- 평점, 내용, 이미지 등록

### 7.2 리뷰 운영 관리

- 노출
- 숨김
- 신고

## 8. 찜 기능

- 숙소 찜 등록
- 찜 해제
- 찜 목록 조회

## 9. 문의 기능

### 9.1 문의방 생성

- 제목, 문의 유형, 관련 숙소/예약 연결

### 9.2 문의 메시지 송수신

- 회원, 판매자, 관리자 간 메시지 송수신
- 읽음 여부 관리

### 9.3 문의 상태 관리

- 접수
- 답변 완료
- 종료
- 차단

## 10. 이벤트/쿠폰 기능

### 10.1 이벤트 관리

- 이벤트 등록/수정/숨김

### 10.2 쿠폰 관리

- 쿠폰 생성
- 발급
- 비활성화

### 10.3 회원 쿠폰 관리

- 발급
- 사용
- 만료
- 취소

## 11. 마일리지 기능

- 적립
- 사용
- 사용취소
- 만료
- 조정

## 12. 관리자 기능

- 판매자 승인 관리
- 회원 상태 관리
- 이벤트/쿠폰 관리
- 문의 모니터링
- 감사 로그 조회
